# Redux Boilerplate
A nice redux setup, with minimal overhead to get started

# get started
```shell
npm i
npm run start
```

# build-scripts

| Script  | Action |
| ------------- | ------------- |
| `start`  | Run Dev Server  |
| `build`  | compile `dist`  |
| `test`  | run unit tests |
| `test:watch`  | run unit tests with watch |
| `clean`  | remove dist |
